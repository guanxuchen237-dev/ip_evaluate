### 5. **Walking on Air or Hopping Mad? Understanding the Impact of Emotions, Sentiments and Reactions on Ratings**

#### 研究背景
- **研究对象**：移动支付App的在线用户评论（Online Customer Reviews, OCR）
- **核心问题**：星级评分是否真实反映了评论文本中的情感？

#### 关键技术与发现
- **三维分析**：
  1. **情感分析（Sentiment Analysis）**：评论文本的整体情感倾向（正面/负面）
  2. **情绪挖掘（Emotion Mining）**：识别具体情绪（快乐、愤怒、悲伤等）
  3. **表情符号分析（Emoji Analysis）**：分析评论中的表情符号

- **研究发现**：
  - **情感和快乐情绪**对星级评分的影响**最强**
  - 其他情绪（愤怒、悲伤）影响较弱
  - 表情符号的影响较小

- **创新点**：
  - 提出了**感知分数（Perception Score）**，比星级评分更准确地反映用户真实感受
  - 公式：`Perception Score = α×Sentiment + β×Happiness + γ×Other Emotions`

#### 对您项目的启发
✅ **评分与情感的关联**：
- 这篇文章证明了**评分不一定完全反映用户真实情感**
- **应用到网文**：
  - 如果网站有评分数据，可以结合评论情感分析验证评分的准确性
  - 如果网站没有评分，**可以用评论情感作为口碑指标**

✅ **多维情感分析**：
- 文章不仅分析整体情感（正面/负面），还分析了具体情绪（快乐、愤怒）
- **进阶方案**：
  - 基础版：只做正面/负面/中性分类
  - 进阶版：识别具体情绪（如"兴奋"、"失望"、"期待"）

✅ **感知分数模型**：
- 可以借鉴这个思路，设计**综合口碑分数**：
  ```python
  口碑分数 = 0.6×情感正面率 + 0.3×评论活跃度 + 0.1×情感强度
  ```

### 6. **Impact of Information Timeliness and Richness on Public Engagement During COVID-19**

#### 研究背景
- **研究主题**：COVID-19期间社交媒体信息的时效性和丰富度对公众参与度的影响
- **数据来源**：中国最大的社交媒体平台（微博）

#### 关键概念
- **信息时效性（Information Timeliness）**的三个维度：
  1. **回顾性（Retrospectiveness）**：回顾过去的信息
  2. **即时性（Immediateness）**：当前正在发生的信息
  3. **前瞻性（Prospectiveness）**：未来展望的信息

- **信息丰富度（Information Richness）**：信息的多样性和详细程度

- **公众参与度（Public Engagement）**：
  1. **广度（Breadth）**：转发数、点赞数
  2. **深度（Depth）**：评论数、讨论深度

#### 研究发现
- **即时性和前瞻性信息**：提高参与度的广度和深度
- **回顾性信息**：降低广度但提高深度（人们会深入讨论历史事件）
- **信息丰富度**：起调节作用（moderating effect）

#### 对您项目的启发
✅ **内容时效性分析**：
- 网络小说也有时效性：
  - **回顾性**：经典老书（如《诛仙》、《盗墓笔记》）
  - **即时性**：正在连载的热门作品
  - **前瞻性**：预告下一章或即将完结

- **可做分析**：
  - 比较"连载中"vs"已完结"作品的IP价值差异
  - 分析更新频率（即时性）对用户互动的影响

✅ **参与度的两个维度**：
- **广度（Breadth）**：点击量、收藏量（覆盖面）
- **深度（Depth）**：评论数、讨论质量（深入程度）

- **应用**：
  - 在您的IP评估中，不仅要看"量"（点击量），也要看"质"（评论深度）
  - 可以设计指标：`参与深度 = 评论数 / 点击量`（评论率）

✅ **NLP技术**：
- 文章使用了**相似度计算方法**来评估信息时效性
- 这是基于NLP的高级技术，可以在您的项目中提及（但不一定要实现）

#### 1. 情感分析技术栈（综合7篇文章）

**基础方案**（适合本科毕设）：
```python
# 使用SnowNLP进行情感分析
from snownlp import SnowNLP

def analyze_comment_sentiment(comment):
    s = SnowNLP(comment)
    sentiment_score = s.sentiments
    
    # 分类
    if sentiment_score > 0.6:
        return '正面', sentiment_score
    elif sentiment_score < 0.4:
        return '负面', sentiment_score
    else:
        return '中性', sentiment_score

# 批量分析
comments = ["这本书真的太好看了！", "剧情太拖沓了", "还可以吧"]
for comment in comments:
    sentiment, score = analyze_comment_sentiment(comment)
    print(f"{comment} → {sentiment} ({score:.2f})")
```

**进阶方案**（如果时间充裕）：
```python
# 使用BERT进行更精准的情感分类
from transformers import BertTokenizer, BertForSequenceClassification
import torch

# 加载预训练的中文情感分类模型
model_name = 'bert-base-chinese'
tokenizer = BertTokenizer.from_pretrained(model_name)
model = BertForSequenceClassification.from_pretrained(model_name, num_labels=3)

def bert_sentiment_analysis(text):
    inputs = tokenizer(text, return_tensors='pt', truncation=True, max_length=512)
    outputs = model(**inputs)
    prediction = torch.argmax(outputs.logits, dim=1).item()
    
    sentiment_map = {0: '负面', 1: '中性', 2: '正面'}
    return sentiment_map[prediction]
```

```python
# 参与广度指标
点击量 = 作品的总浏览次数
收藏量 = 加入书架的用户数
覆盖面指数 = (点击量 + 收藏量×10) / 平均值

评论数 = 用户评论总数
评论率 = 评论数 / 点击量
打赏活跃度 = 打赏人数 / 收藏量

# 综合参与度
用户参与度 = 0.6×覆盖面指数 + 0.4×参与深度
```
