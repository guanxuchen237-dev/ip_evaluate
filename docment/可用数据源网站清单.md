## 🛠️ 爬取技术方案

### 基础爬虫模板（Scrapy）

```python
import scrapy
from scrapy import Request

class NovelSpider(scrapy.Spider):
    name = 'novel_spider'
    
    # 纵横中文网示例
    start_urls = ['http://www.zongheng.com/store/c0/c0/b0/u0/p1/v0/s1/t0/u0/i1/ALL.html']
    
    custom_settings = {
        'DOWNLOAD_DELAY': 2,  # 下载延迟2秒
        'CONCURRENT_REQUESTS': 1,  # 并发请求数
        'USER_AGENT': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }
    
    def parse(self, response):
        # 提取作品列表
        for novel in response.css('.book-item'):
            novel_url = novel.css('a::attr(href)').get()
            yield Request(url=novel_url, callback=self.parse_detail)
        
        # 翻页
        next_page = response.css('.next::attr(href)').get()
        if next_page:
            yield Request(url=next_page, callback=self.parse)
    
    def parse_detail(self, response):
        # 提取详细信息
        yield {
            '作品名': response.css('.book-name::text').get(),
            '作者': response.css('.author::text').get(),
            '点击量': response.css('.click-count::text').get(),
            '收藏量': response.css('.collect-count::text').get(),
            # ... 其他字段
        }
```

### 反爬虫中间件

```python
# middlewares.py
import random
from fake_useragent import UserAgent

class RandomUserAgentMiddleware:
    def __init__(self):
        self.ua = UserAgent()
    
    def process_request(self, request, spider):
        request.headers['User-Agent'] = self.ua.random

class ProxyMiddleware:
    def __init__(self):
        self.proxies = [
            'http://proxy1.com:8080',
            'http://proxy2.com:8080',
            # 代理池列表
        ]
    
    def process_request(self, request, spider):
        request.meta['proxy'] = random.choice(self.proxies)
```

## ⚠️ 法律与伦理提醒

### 合规性原则
1. **遵守robots.txt**：检查网站的爬虫协议
   ```python
   # 检查方法
   访问：http://www.zongheng.com/robots.txt
   ```

2. **控制爬取频率**：
   - 建议每秒不超过1次请求
   - 使用`DOWNLOAD_DELAY = 2`（Scrapy设置）

3. **仅用于学术研究**：
   - 不得商用
   - 不得二次传播数据
   - 论文中说明数据来源

4. **尊重版权**：
   - 不爬取VIP付费章节内容
   - 不爬取完整章节文本（仅爬元数据）

### 风险提示
- ⚠️ IP被封：准备备用代理
- ⚠️ 账号被封：不要使用个人常用账号
- ⚠️ 法律风险：避免大规模商业化使用
