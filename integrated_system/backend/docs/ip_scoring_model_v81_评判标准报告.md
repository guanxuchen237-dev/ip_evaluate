# IP评分模型 v8.1 评判标准报告

## 一、模型概述

**版本**: v8.1  
**创建时间**: 2026-03-15  
**核心原则**: 月票是商业价值的核心指标，排名决定等级上限，转化率反映付费意愿

---

## 二、等级判定规则

### 2.1 排名等级（决定等级上限）

| 排名范围 | 等级 | 基础分 |
|----------|------|--------|
| 第1-10名 | S级候选 | 90分 |
| 第11-50名 | A级候选 | 80分 |
| 第51-100名 | B级候选 | 70分 |
| 第101-200名 | C级候选 | 60分 |
| 第201名以后 | D级候选 | 50分 |

### 2.2 月票等级（决定等级下限）

#### 起点平台

| 月票数量 | 等级 |
|----------|------|
| ≥10000票 | S级 |
| ≥5000票 | A级 |
| ≥2000票 | B级 |
| ≥1000票 | C级 |
| ≥500票 | D级 |
| <500票 | E级（极差）|

#### 纵横平台（月票普遍较低，阈值调整）

| 月票数量 | 等级 |
|----------|------|
| ≥3000票 | S级 |
| ≥1500票 | A级 |
| ≥800票 | B级 |
| ≥400票 | C级 |
| ≥200票 | D级 |
| <200票 | E级（极差）|

### 2.3 最终等级计算

```
最终等级 = min(排名等级, 月票等级)
```

**示例**：
- 排名第5名（S级候选）+ 月票3000票（A级）→ 最终A级
- 排名第33名（A级候选）+ 月票345票（D级）→ 最终D级
- 排名第500名（D级候选）+ 月票10000票（S级）→ 最终D级

## 三、转化率评分

### 3.1 转化率计算

```
转化率 = 月票数 / 总推荐数
```

### 3.2 转化率评级（区分平台）

#### 起点平台

| 转化率 | 评级 | 加分 |
|--------|------|------|
| ≥5% | 优秀 | +5分 |
| ≥2% | 良好 | +3分 |
| ≥1% | 一般 | +1分 |
| ≥0.5% | 较差 | +0分 |
| <0.5% | 极差 | -3分 |

#### 纵横平台

| 转化率 | 评级 | 加分 |
|--------|------|------|
| ≥3% | 优秀 | +5分 |
| ≥1% | 良好 | +3分 |
| ≥0.5% | 一般 | +1分 |
| ≥0.2% | 较差 | +0分 |
| <0.2% | 极差 | -2分 |

## 五、最终评分计算

### 5.1 计算公式

```
最终评分 = 基础分 + 转化率加成 + 字数加成 + 点击加成
```

### 5.2 分数范围限制

| 等级 | 分数范围 |
|------|----------|
| S级 | 90-99分 |
| A级 | 80-89分 |
| B级 | 70-79分 |
| C级 | 60-69分 |

| E级 | 40-49分 |

## 八、平台差异说明

### 8.1 起点 vs 纵横

| 维度 | 起点 | 纵横 |
|------|------|------|
| 月票水平 | 高（头部可达10万+）| 低（头部通常3000-5000）|
| 月票阈值 | 标准阈值 | 约为起点的30% |
| 转化率基准 | 1%为一般 | 0.5%为一般 |
| 读者付费意愿 | 较强 | 较弱 |

### 8.2 平台识别

模型会根据 `platform` 字段自动选择对应的阈值标准：
- `Qidian` → 起点标准
- `Zongheng` → 纵横标准

---

## 九、模型使用说明

### 9.1 输入参数

```python
book_data = {
    'title': '书名',
    'rank': 排名,                    # 必填
    'monthly_tickets': 月票数,        # 必填
    'total_recommend': 总推荐数,      # 必填
    'word_count': 字数,              # 可选
    'total_click': 总点击,           # 可选
    'platform': 'Qidian/Zongheng'    # 必填
}
```

### 9.2 输出结果

```python
{
    'ip_score': 评分,           # 40-99
    'ip_grade': 等级,           # S/A/B/C/D/E
    'rank_tier': 排名等级,      # S/A/B/C/D
    'ticket_tier': 月票等级,    # S/A/B/C/D/E
    'conversion_rate': 转化率,  # 0-1
    'conversion_level': 转化评级,
    'verdict': 好书判定,
    'commercial': 商业价值评估
}
```
